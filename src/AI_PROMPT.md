# "무제" 소셜 미디어 플랫폼 구축 프롬프트

일기 형태의 개인적인 이야기를 공유하는 소셜 미디어 플랫폼 "무제"를 구축해주세요.

## 플랫폼 컨셉
- 사용자들이 취업, 연애, 가족, 정신건강 등의 고민을 카테고리별로 나누어 공유하고 서로 공감할 수 있는 따뜻한 커뮤니티
- 😢 걱정과 불안 탭과 💛 감사와 따뜻함 탭으로 피드 구분
- 종이책 질감의 따뜻한 미색 톤 디자인
- 모바일 최적화 (하단 네비게이션 + 우측 하단 floating 버튼)

## UI/UX 디자인

### 컬러 시스템
- 전체 배경: 그라데이션 `linear-gradient(135deg, #faf8f3 0%, #f8f6f0 50%, #faf8f3 100%)`
- 카드 배경: `#f5f3ed` (따뜻한 미색)
- 카드 테두리: `#e8e6e0`
- 프로필 박스 배경: `#ede8dc` (더 어두운 톤)
- 격려 카드: `from-amber-50 to-orange-50` 그라데이션, `border-amber-200/50`

### 레이아웃
- 모바일 최적화 반응형 디자인
- 최대 너비: `max-w-4xl` (피드 영역)
- 카드 스타일: 둥근 모서리, hover 시 그림자 강조

## 핵심 기능

### 1. 헤더
- 왼쪽: 응원 스티커 개수 표시 (Sparkles 아이콘 + 숫자), 툴팁으로 "{사용자명} 님이 보낼 수 있는 응원 스티커 개수는 {개수} 개 입니다." 표시
- 중앙: Heart 아이콘 + "무제" 로고
- 오른쪽: 알림 벨 아이콘 (미읽음 알림 개수 뱃지 표시)
- Sticky 헤더 (스크롤 시 상단 고정)

### 2. 피드 탭 시스템 (5개 탭)
1. **😢 걱정과 불안 탭** (worry)
   - 제목: "😢 걱정과 불안"
   - 설명: "당신의 걱정을 나누세요. 당신만 그런 게 아니에요."
   - feedType이 "worry"인 게시글만 표시

2. **💛 감사와 따뜻함 탭** (grateful)
   - 제목: "💛 감사와 따뜻함"
   - 설명: "따뜻했던 순간을 나누세요. 당신의 이야기가 누군가에게는 힘이 돼요."
   - feedType이 "grateful"인 게시글만 표시

3. **공감한 이야기 탭** (empathy)
   - Heart 아이콘
   - 제목: "공감한 이야기"
   - 설명: "당신이 공감한 이야기들 - 당신만 그런 게 아니에요."
   - 현재 사용자가 공감한 게시글만 표시
   - 게시글이 없으면 빈 상태 메시지 표시

4. **프로필 탭** (profile)
   - 👤 아이콘
   - 프로필 카드 (배경색: `#ede8dc`)
     - 아바타 (28x28, 둥근 모양)
     - 닉네임, 도시, 나이대, 직업 정보
     - 프로필 편집 버튼 (Pencil 아이콘)
   - 내가 작성한 게시글 목록

5. **설정 탭** (settings)
   - ⚙️ 아이콘
   - 텍스트 크기 조절 (14px ~ 22px, 슬라이더)
   - 미리보기 영역
   - 앱 정보 (버전 1.0.0, 개발자: 무제 팀)

### 3. 카테고리 필터
- 전체, 커리어, 이직, 취직, 연애, 결혼, 육아, 가족 관계, 건강, 재정, 인간 관계, 퇴직, 학교 생활, 학업, 직장 생활, 진로, 우울증
- 배지 형태, 클릭 시 토글
- 선택된 카테고리만 필터링 (복수 선택 가능)

### 4. 게시글 카드
- **헤더**
  - 아바타 (12x12, 둥근 모양, 그라데이션 fallback: `from-blue-400 to-purple-500`)
  - 닉네임
  - 도시 · 나이대 직업 (작은 회색 텍스트)
  - 시간 표시 (Clock 아이콘 + "방금 전", "1시간 전", "N시간 전", "1일 전", "N일 전")

- **본문**
  - 게시글 내용 (설정에서 조절 가능한 폰트 크기)
  - 카테고리 배지들

- **응원 스티커 표시**
  - 스티커가 있으면 표시 (최대 3개 이모지 보여주고 나머지는 +N)
  - 클릭 시 팝오버로 모든 스티커 상세 보기

- **액션 버튼**
  - 공감 버튼: Heart 아이콘 + "N명이 공감해요" (클릭 시 토글, 공감하면 하트 채워짐)
  - 응원 스티커 보내기: Sparkles 아이콘 + "응원하기" (본인 게시글이거나 이미 보낸 경우 비활성화)

### 5. 격려 카드
- 8개 게시글마다 격려 문구 카드 삽입
- amber 톤 그라데이션 배경
- Sparkles 아이콘 + 격려 문구 (이탤릭체)
- 격려 문구 5개:
  1. "잘 할 수 있고 잘 할 거예요. 시간이 더 필요할 뿐이에요."
  2. "사소한 일들이 밀려와도 괜찮아요. 모두들 그러하듯 견딜 수 있어요."
  3. "후회만 가득한 과거와 불안하기만 한 미래 때문에 지금을 망치지 마세요. 오늘을 살아가세요."
  4. "마음에 있는 불꽃이 꺼지는 일이 없을 거예요."
  5. "인생은 때때로 힘들지만, 그 속에서 무엇을 해낼 것인지는 우리 스스로의 선택이에요."

### 6. 게시글 작성 (Floating Button)
- 우측 하단 고정 (bottom-20 right-4)
- Edit3 아이콘, 둥근 버튼 (14x14)
- 클릭 시 다이얼로그 오픈

**다이얼로그 구성:**
- 피드 타입 선택: "😢 걱정/불안" 또는 "💛 감사/따뜻함"
- 텍스트 영역 (최소 높이 200px)
  - worry: "오늘 나는 걱정이 되는 게... / 요즘 힘든 게... / 불안한 일이..."
  - grateful: "오늘 감사했던 일은... / 따뜻했던 순간... / 행복했던 경험..."
- 글자 수 표시
- 카테고리 선택 (최소 1개 필수, 배지 클릭 토글)
- 취소/공유하기 버튼

### 7. 공감 기능
- 게시글에 공감 버튼 클릭 시:
  - 공감 수 증가/감소 (토글)
  - 하트 아이콘 채우기/비우기
  - 본인 글이 아닌 경우 작성자에게 알림 전송

### 8. 응원 스티커 시스템
- 스티커 6종류:
  - 💪 힘내세요!
  - 🌟 응원해요!
  - 🤝 함께 있어요
  - 💚 괜찮아요
  - 👏 잘하고 있어요!
  - ☀️ 힘을 내요!

**스티커 보내기 로직:**
- 다이얼로그에서 스티커 선택
- 스티커 개수가 0개면 전송 불가
- 이미 해당 게시글에 보낸 경우 전송 불가
- 본인 게시글에는 전송 불가
- 전송 시 스티커 개수 -1
- 특수 케이스: 본인 글에 보낼 경우 "익명의 친구"가 보낸 것으로 시뮬레이션 (스티커 개수 +1)
- 작성자에게 알림 전송

### 9. 알림 시스템
- 우측 Sheet 패널
- 알림 타입 2가지:
  1. **공감 알림**: "{사용자명}님이 회원님의 이야기에 공감했습니다"
  2. **스티커 알림**: "{사용자명}님이 응원을 보냈어요!" (스티커 이모지 표시)
- 알림 카드:
  - 아바타 + 사용자명
  - 시간 (방금 전, N시간 전 등)
  - 게시글 내용 미리보기 (2줄 제한)
- 미읽음 알림은 강조 배경 (`bg-accent/50`)
- 알림창 열면 모든 알림 자동 읽음 처리
- 읽지 않은 알림 개수 뱃지 표시 (9+ 초과 시 "9+" 표시)

### 10. 프로필 편집
- 다이얼로그 형태
- 닉네임 입력 (최대 20자, 글자 수 표시)
- 프로필 사진 URL 입력
- 미리보기 영역 (아바타 + 닉네임 + 위치 정보)
- 저장 시 모든 게시글의 사용자 정보도 업데이트

### 11. 하단 네비게이션
- 5개 탭:
  1. 🌧️ (걱정과 불안)
  2. ☀️ (감사와 따뜻함)
  3. Heart 아이콘 (공감한 이야기)
  4. 👤 (프로필)
  5. Settings 아이콘 (설정)
- 선택된 탭은 배경색 강조 (`bg-accent`)
- 고정 위치 (sticky bottom)

```typescript
{
  id: string;

  avatar: string;
  bio: string;
  city: string;
  ageGroup: string;
  occupation: string;
```

```typescript
{
  id: string;
  userId: string;

  userAvatar: string;

  userAgeGroup: string;

  feedType: "worry" | "grateful";

  empathyCount: number;

```

```typescript
{
  id: string;
  type: "empathy" | "sticker";

  fromUserAvatar: string;
  storyId: string;
  storyContent: string;
  stickerEmoji?: string;
  stickerMessage?: string;
  createdAt: Date;
  read: boolean;
}
```

## 기술 스택
- React
- TypeScript
- Tailwind CSS v4
- Lucide React (아이콘)
- shadcn/ui 컴포넌트 (Dialog, Sheet, Tabs, Card, Button, Avatar, Badge, Slider 등)

## 추가 요구사항
- 모든 텍스트는 한국어
- 시간 표시는 한국어 ("방금 전", "N시간 전", "N일 전")
- 따뜻하고 공감하는 분위기의 UI
- 반응형 디자인 (모바일 우선)
- 모든 상태는 클라이언트에서 관리 (useState 사용)
- 프로필 업데이트 시 기존 게시글의 사용자 정보도 함께 업데이트
- 폰트 크기 설정은 게시글 본문과 프로필의 내 게시글에 모두 적용

## 컴포넌트 구조

```
App.tsx (메인)

├── CategoryFilter.tsx (카테고리 필터)

│   └── MyStoryCard.tsx (내 게시글 카드)
└── Settings.tsx (설정 페이지)
```
